¿Que es un framework?

Los framework CSS son herramientas utilizadas por los desarrolladores de interfaz de usuario para facilitar su trabajo. Los framework brindan a los desarrolladores las herramientas para acelerar rápidamente las interfaces de usuario que se pueden modificar e iterar a lo largo de un proyecto en lugar de perder tiempo comenzando desde un documento en blanco.

En su forma más simple, un framework CSS es una colección de hojas de estilo CSS que están preparadas y listas para usar. Están diseñados para su uso en situaciones comunes, como la configuración de barras de navegación, y a menudo se amplían con otras tecnologías como SASS y JavaScript.

Bulma

¿Quien lo creo? ¿De donde surge?

Bulma es un framework CSS de código abierto desarrollado por Jeremy Thomas lanzado por primera vez en 2016. Este marco se basa en la propiedad CSS Flexbox. El cual lo hace altamente receptivo (Responsive).

Bulma ha ganado terreno entre los desarrolladores front-end, lo suficiente como para competir con otros framework CSS conocidos como Bootstrap.

Para usar bulma sólo necesitas un archivo CSS para comenzar a utilizarlo en tus proyectos o puedes optar por la versión en la que descargas los archivos Sass y modificar las variables a tu gusto.

En Bulma no existe ningún archivo JavaScript, así que lo puedes usar tranquilamente sin necesidad de ninguna librería o plugin extra.
